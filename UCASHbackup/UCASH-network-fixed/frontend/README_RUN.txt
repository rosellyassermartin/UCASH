UCash Frontend Ready
====================

This zip includes the missing files needed to run the frontend:
- package.json
- vite.config.js
- index.html
- src/main.jsx
- src/App.jsx
- src/styles.css

It also includes your uploaded student page components and api.js.

How to run:
1. Open terminal in this frontend folder
2. Run: npm install
3. Run: npm run dev
4. Open: http://localhost:5173

Note:
- This frontend uses sample navigation and a sample student user.
- Several page components still use hardcoded demo data, so the UI will open,
  but not every screen is fully connected to the backend yet.
